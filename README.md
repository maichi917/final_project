# my stocker（仮）

##  サービス概要
日用品のストック管理アプリ
面倒なストック管理をスマホで全部確認できるようにして、うっかり切らしたり買いすぎたりすることを防ぐ。
カテゴリ分け（ヘアケア、化粧品など）し、どこにどのくらいお金を使っているのか可視化する。
また、使い終わった後に五段階で評価をして、ひとことコメントをつけ次回以降の買い物の参考とする。
ex.「★★★☆☆ 使い心地がいいがコスパが悪い」

## 開発背景
シャンプーやトリートメントなどのヘアケア用品から歯ブラシなどの日用品、化粧品に至るまで、気に入ったものをまとめ買いすることが多い。
ヘアケア用品に関しては、いつも大容量の詰め替え用を買っている。（1000mlで2万円近くのものもある）
ストックがあると思っていたのになくて困ったり、在庫を把握していないために急に大きな出費が発生するのが悩みだった。
加えて、面倒くさがりな性格であるため、スマホひとつで寝ながらでも、出先で思い立った時でもストックBOXに現物を実際に見にいかずして在庫管理と発注がしたい。
また、色々使うのが好きなので、同時に使用感や感想などをストックできたら嬉しい。

## メインターゲット層
自分のような面倒くさがりだけど色々使いたいこだわり派の女性。

## サブターゲット層
家庭で家事を担っている人。（たくさんの種類の日用品を管理する必要があるため）

### ターゲット層の理由
メイン機能がストック管理であるため、物を管理したい人がターゲットとなる。
加えて、アイテムを評価することで次回以降の買い物の参考にできるアプリを作ることで、日常を楽しくしたい。

### ユーザー獲得方法
XやInstagramなどのSNSでシェア

## サービス利用のイメージ
すきま時間にスマホでサクッと在庫管理。
使い終わりのタイミングで簡単な評価をし、次回以降購入するかどうかの判断材料にもできる。
ストック管理をしながらお気に入りリストができる。

### サービス差別化のポイント・推しポイント
類似サービス「monoca」「zaico」
細かくたくさんの物を管理するためのアプリはたくさんあるが、あまり複雑化すると登録や管理自体が億劫になる。
本アプリでは、登録さえしてしまえば「使いはじめ」「使い切り」ボタンだけで自動で使用期間を算定し、次回以降のなくなるタイミングのデータとすることができる。
また、ストック管理だけでなくお気に入りアイテムのデータを蓄積していくことができる。

## 機能の実装方針予定
### 1. 使用期間の自動計算と予測
- 「使い始め日」と「使い切り日」をデータベースに記録
- 使用日数 = 使い切り日 - 使い始め日
- 同じ商品の過去の使用履歴から平均使用日数を計算
- 予測日 = 現在の使い始め日 + 平均使用日数

### 2. なくなりそうな商品の上位表示
- 「予測日 - 今日」で「あと何日でなくなるか」を計算
- この値が小さい順にソートして表示

### 3. 購入リマインド機能(本リリース)
- なくなる予測日の2週間前にメール通知
